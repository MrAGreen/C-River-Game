\doxysubsection{Detailed Description}
\doxylink{class_game}{Game} class for managing river trivia gameplay. 

This class represents the game state and logic for a river trivia game. It interacts with a {\ttfamily \doxylink{class_rivers}{Rivers}} object for river data access and manages user input, round execution, scoring, and fastest time tracking.

Retrieves and formats the top 5 fastest round scores. 

This function retrieves the top 5 fastest rounds from the {\ttfamily fastest\+Times} vector, which stores pairs of time and river information for successful rounds. It performs\+:
\begin{DoxyEnumerate}
\item Sorts the {\ttfamily fastest\+Times} vector in ascending order of time using {\ttfamily Sort\+Duration}.
\item Iterates through the sorted vector and extracts the second element of each pair, which contains formatted information about the rivers and time for that round.
\item Stores those formatted strings in a new vector {\ttfamily top\+Five}.
\item Resizes {\ttfamily top\+Five} to a maximum of 5 elements to ensure only the top 5 scores are returned.
\item Returns the {\ttfamily top\+Five} vector containing strings representing the top 5 fastest rounds.
\end{DoxyEnumerate}

{\footnotesize\ttfamily std\+::string Game\+::point\+Scoring (\begin{DoxyParamCaption}\item[{std\+::string}]{input,  }\item[{bool}]{b }\end{DoxyParamCaption})}



Provides feedback and scoring based on user input, game mode, and time limit. 

This function evaluates the user\textquotesingle{}s input ({\ttfamily input}), a boolean flag ({\ttfamily b}) indicating the game mode (true for same continent, false for different continents), and the measured time ({\ttfamily dur}). It provides feedback messages accordingly\+:


\begin{DoxyItemize}
\item {\bfseries{Time Limit Exceeded\+:}} If the time taken ({\ttfamily dur}) exceeds 10 seconds (10000 milliseconds), it returns an "{}\+Incorrect\+: time limit exceeded"{} message.
\item {\bfseries{Correct Answers\+:}}
\item If the user input ({\ttfamily input}) is "{}d"{} (different) and the rivers are indeed from different continents ({\ttfamily b} is false), it adds the time to {\ttfamily fastest\+Times} using {\ttfamily add\+Time} and returns a success message with the continent information for both rivers.
\item If the user input ({\ttfamily input}) is "{}s"{} (same) and the rivers are from the same continent ({\ttfamily b} is true), it adds the time to {\ttfamily fastest\+Times} using {\ttfamily add\+Time} and returns a success message with the shared continent information.
\item {\bfseries{Incorrect Answers\+:}}
\item If the user input ({\ttfamily input}) is "{}d"{} (different) but the rivers are from the same continent ({\ttfamily b} is true), it returns an incorrect message with the shared continent information.
\item If the user input ({\ttfamily input}) is "{}s"{} (same) but the rivers are from different continents ({\ttfamily b} is false), it returns an incorrect message with the continent information for both rivers.
\end{DoxyItemize}
